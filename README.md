# Stock Price Fetcher

This project downloads the last 30 days of daily prices for Apple (AAPL) using [yfinance](https://pypi.org/project/yfinance/) and prints the most recent closing price.

## Installation

It is recommended to use a virtual environment. Then install the dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the script with:

```bash
python main.py
```
